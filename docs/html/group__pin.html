◆ pin_mode_t

enum pin_mode_t

Pin in/out modes enum.

Enumerator
QUASI_BIDIRECTIONAL 

Quasi bidirectional mode.

Pin can be used for read and write digital data. Sink Current up to 20mA, pull-up Current is 270μA. Because of manufactured error, the actual pull-up current is 270uA ~ 150uA

PUSH_PULL 

Strong pull-up output.

Current can be up to 20mA, resistors need to be added to restrict current

INPUT_ONLY 

input-only (high-impedance)

OPEN_DRAIN 

Open drain mode.

Internal pull-up resistors should be disabled and external pull-up resistors need to join
