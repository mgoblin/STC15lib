Detailed Description

Example demonstrating conversion from milliseconds to timer0 ticks.

This example shows how to convert a time duration in milliseconds to the corresponding number of timer ticks for Timer0 in Mode 0 (13-bit timer mode).

The example configures Timer0 to run in 1T mode (one clock cycle per count) and calculates the number of ticks needed for a 3ms delay using the timer0_ms_to_ticks() function. It then starts the timer and outputs the calculated tick value over UART.

Note
The actual tick value will depend on the system clock frequency defined in sys.h (MAIN_Fosc). With the default 11.0592MHz clock, 3ms corresponds to approximately 33,177 ticks.
