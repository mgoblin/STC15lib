#define timer0_mode1_Hz_to_ticks (   frequency)    (timer_Hz_to_ticks(frequency, get_timer0_clock_divider(), get_frequency_divider_scale()))

Converts frequency to ticks for timer0 mode1.

Call this method after timer0 is initialized and MCU clock frequency divider scale is set.

Warning
This method returns 0 on ticks overflow of result. Be careful.
Parameters
frequencyuint32_t frequency value in Hz. Should not be 0;
Returns
uint16_t timer0 ticks corresponding to frequency value or 0 on uint16_t overflow

◆ timer0_mode1_ticks_to_Hz

#define timer0_mode1_ticks_to_Hz (   ticks)    (timer_ticks_to_Hz(ticks, get_timer0_clock_divider(), get_frequency_divider_scale()))

Converts ticks to timer overflow frequency for timer0 mode1.

The output frequency = (SYSclk/timer_clock_divider)/(1 + ticks).

Call this method after timer0 is initialized and MCU clock frequency divider scale is set.

uint16 ticks used in timer mode 0, 1, 3. For timer mode2 (8 bit autoreladable timer) ticks can be in 0..255 range.

Parameters
ticksuint16_t timer ticks count to covert
Returns
float frequency multiplied by 100 corresponding to ticks count
