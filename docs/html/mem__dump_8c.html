Detailed Description

Dump IDATA and XDATA memory contents to UART.

Dumps the first 256 bytes of IDATA (internal RAM) and the first 256 bytes of XDATA (external RAM) to UART1 at 115200 baud.

Output format is a standard hex dump with 16 bytes per row, showing offset addresses and byte values in hexadecimal.
