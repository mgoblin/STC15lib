#define wdt_init (   wdt_scale)
Value:
{ \
WDT_CONTR &= 0x00; \
WDT_CONTR |= (wdt_scale & 0x07); \
}

Init WDT.

WDT overflow time = (12 * wdt_scale * 32768) / MainFosc

Parameters
wdt_scaleuint8_t WDT timer scale in range 0..7
