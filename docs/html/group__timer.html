Detailed Description

Functions and data structures for timers.

This module supports two generar purpose timers Timer0 and Timer1. Each timer can work as COUNTER or TIMER. Counter increments on external singnal (like pin 1/0 change state). Input of timer is system clock.

8051 chips timer has 4 modes

  • mode0 - 8-bit timer/counter with 5-bit prescaler (13 bit timer)
  • mode1 - 16-bit timer/counter
  • mode2 - 8-bit auto reload timer/counter
  • mode3 - The function depends on Timer0 or Timer1

This module version supports only mode0 for timer. Other timer modes and counter mode does not implements yet. Timer interrups does not implements yet.
