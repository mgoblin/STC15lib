Detailed Description

Functions and data structures for UART1 communication in Mode 0.

Mode 0 provides synchronous communication with fixed baud rate.

Receives a single byte from UART1.

Waits until a byte is received via UART1 and stores it in the provided memory location.

Parameters
[out]bytePointer to a uint8_t where the received data will be stored.
Note
This function blocks until a byte is received.
Requires UART1 to be initialized with uart_mode0_init() prior to use.
Assumes UART1 is configured with 8 data bits, no parity, 1 stop bit.
