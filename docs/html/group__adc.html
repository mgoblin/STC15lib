ADC initialization function.

Configures the ADC for 10-bit resolution, uses the internal voltage reference.

enum adc_pin_mode_t

ADC P1 modes enum.

describes P1 pin modes for ADC

ADC can be used only with P1 pins. Each P1 pin can be configured as ADC input. P1 pins can be configured as input only or open-drain.
